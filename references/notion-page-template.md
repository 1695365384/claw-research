# Notion Page Template (Reference)

> ⚠️ This is a **reference file**. The core template is in `SKILL.md`.

## Report Structure

1. Executive Summary - Conclusion + Actions + Risks
2. Bayesian Scores - Probability table
3. Top Pain Points - Evidence blocks
4. Strategic Analysis - SWOT + Competitors
5. Action Items - Checklist
6. Notion Sync - Status

## Evidence Block Format

```markdown
> **原文**: "[Complete original text]"
> **来源**: [Source Name](URL)
> **类型**: community_discussion
> **发布时间**: YYYY-MM-DD
```

## Source Types

| Type | Examples |
|------|----------|
| `community_discussion` | Reddit, HN, 知乎 |
| `blog_article` | Medium, Substack |
| `industry_survey` | User Interviews |
| `competitor_review` | G2, Capterra |
| `social_post` | Twitter/X, LinkedIn |
